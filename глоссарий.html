Алгоритм - описание последовательности шагов в решении задачи, приводящих от исходных данных к требуемому результату за конечное число шагов.

Алгоритмический язык - формальный язык, предназначенный для записи алгоритмов

Алфавит - набор используемых символов

Блок-схема - графическое изображение, дающее представление о порядке работы алгоритма

Величина - отдельный информационный объект (число, строка, символ)

Ветвление - алгоритмическая контсрукция, в которой в зависимости от результата проверки условия предусмотрен выбор одной из двух последовательностей действий

Выражение - языковая конструкция для вычисления значения с использованием одного или нескольких операндов

Дискретность - разделение процесса решения задачи на отдельные шаги, которые выполняются исполнителем последовательно

Имя - условное обозначение некой величины

Исполнитель - некоторый объект, способный выполнять определенный набор команд

Константа - величина, значение которой указывается в тексте алгоритма и не меняется в процессе его исполнения

Линейный алгоритм - алгоритм, в котором используется только конструкция "следование"

Массив (см. таблица)

Операнд - объект над которым выполняются операции

Переменная - величина, значение которой меняется в процессе исполнения алгоритма

Повторение - алгоримическая конструкция, представляющая собой последовательность действий, выполняемых многократно

Построчная запись - запись алгоритма на естественном языке с соблюдением дополнительных правил
1. каждое предписание записывается с новой строки;
2. шаги алгоритма нумеруются;
3. исполнение алгоритма происходит в порядке возрастания номеров.

Разветвляющийся алгоритм - алгоритм, в основе которого лежит структура "ветвление"

Семантика - система правил, строго определяющих смысл и способ употребления конструкций языка

Синтаксис - система правил, по которым из символов алфавита образуются правильные конструкции языка

Следование - алгоритмическая конструкция, отображающая естественный, последовательный порядок действий

Словесное описание - самый простой способ записи алгоритма на естественном языке

Таблица - набор некоторого числа однотипных элементов, которым присвоено одно имя

Тело цикла - последовательность действий, многократно повторяющаяся в процессе выполнения цикла

Тип - определитель множества допустимых значений величины и множества применимых операций к ней

Циклический алгоритм - алгоритм, содержащий конструкцию повторения
