Сайт разработан для изучения учащимися 8 классов темы "Основы алгоритмизации". Его содержание отвечает требованиям Федеральной образовательной программы и базируется на содержании УМК "Информатика" под редакцией Л.Л. Босовой.

В курсе имеется четыре раздела, каждый из которых разделен на 3 блока: теоретический, практический и контрольный.

По мимо этого на сайте имеется итоговый контрольный тест, глоссарий и список литературы для самостоятельного углубленного изучения.

Содержание курса расчитано на 7 академических часов и может использоваться как для работы в классе, так и для самостоятельного изучения.

В ходе изучения учащимися раздела решаются следующие задачи:

    Понимать смысл понятия «алгоритм» и области его применения;
    Знать основные свойства алгоритма;
    Уметь исполнять алгоритм для формального исполнителя с заданной системой команд;
    Знать различные способы записи алгоритмов;
    Уметь определять преимущества и недостатки различных форм записи;
    Иметь представление о величинах, с которыми работают алгоритмы;
    Понимать смысл операции присваивания;
    Знать и уметь применять на практике правила записи выражений;
    Обобщить представления об алгоритмической конструкции «следование»;
    Уметь разрабатывать и выполнять линейные алгоритмы для различных исполнителей;
    Иметь представления об алгоритмической конструкции «ветвление»;
    Уметь записывать составные условия для алгоритма, выполнять его;
    Аргументированно выбирать форму ветвления для решения поставленной задачи;
    Иметь представления об алгоритмической конструкции «цикл»;
    Знать назначение конструкции «цикл с заданным условием продолжения работы», уметь применять его для решения учебной задачи;
    Знать назначение конструкции «цикл с заданным условием окончания работы»;
    Уметь составлять простые циклические алгоритмы для формального исполнителя;
    Уметь записывать цикл с заданным числом повторений, самостоятельно определять алгоритмическую конструкцию для решения поставленной задачи;
