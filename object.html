Величины

Любой алгоритм действует не сам по себе - он производит действия над некоторыми величинами.

Величина - отдельный информационный объект (число, строка, таблица)

Постоянная величина (константа) задается в тексте алгоритма и не меняется.
Переменная же имеет определенное имя, а ее значение может изменяться в процессе исполнения алгоритма.

Над величинами алгоритма выполняются различные операции:
1. арифметические (+, -, *, /);
2. отношения (<, >, =);
3. логические (и, или, не).

Операнд - объект над которым выполняются операции

Не каждый объект может быть операндом для операции. Например, текст не может быть объектом для арифметических операций.

Множество величин, объединенных множеством допустимых значений и определенной совокпностью допустимых операций, называют величинами определенного типа.

Значением символьного типа является один символ: буква, цифра или другой символ. Для литерального типа характерно использование последовательности символов, их значение указывается в кавычках. Величины логического типа могут принимать только два значения (истина или ложь).

Для ссылок на величины используются их имена, которые обозначаются латинскими буквами и цифрами (a, b1, sum).

Выражения

Выражение - языковая конструкция для вычисления значения с использованием одного или нескольких операндов

Выражения состоят из операндов, объединенных знаками операций. Выражения записываются в виде линейных последовательностей символов (без подстрочных и надстрочных символов, обыкновенных дробей). Порядок выполнения операций определяется их приоритетом и скобками. Операции одинакового приоритета выполняются слева направо.

!!! Знаки операций пропускать нельзя !!!

Команда присваивания

Команда присваивания позволяет задать конкретное значение величины. В общем виде присваивание имеет вид: имя переменной := выражение

Например А:=В+5 читается как: "переменной А присвоить значение выражения В плюс 5". При выполнении команды присваивания сначала вычисляется значение выражения справа, а затем результат присваивания переменной.

Знаки = и := являются разными

Свойства присваивания:
1.пока переменной не присвоено значение она остается неопределенной;
2. значение, присвоено переменной, сохраняется в ней до выполнения следующего присваивания этой переменной нового значения;
3. если переменной присваивается новое значение, то предыдущее безвозвратно теряется.

Табличные величины

В повседневной жизни мы часто сталкиваемся с различными таблицами (список класса, расписание уроков и т.д.)

Линейная таблица (одномерный массив) состоит из однотипных данных, записанных в одну линию или столбик.

Прямоугольная таблица (двумерный массив) - упорядоченный некоторым образом набор строк (столбцов), содержащих одинаковое количество элементов. Строки и столбцы такой таблицы имеют свою нумерацию.

Всей совокупности элементов табличной величины дается одно имя, а элементы различаются по номерам, называемым индексами.
